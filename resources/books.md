# Recommended Books for AI Engineers

Curated reading list organized by phase — prioritized by impact per hour spent.

## Phase 0–2: Foundations

| Book | Author | Why Read It | Free? |
|---|---|---|---|
| **Automate the Boring Stuff with Python** | Al Sweigart | Best beginner Python book; practical projects | ✅ Free online |
| **Python Tricks** | Dan Bader | Advanced Python idioms and best practices | ❌ |
| **Fluent Python** | Luciano Ramalho | Deep Python mastery — generators, protocols, OOP | ❌ |
| **Clean Code** | Robert C. Martin | Writing readable, maintainable code | ❌ |

### Mathematics

| Book | Author | Why Read It | Free? |
|---|---|---|---|
| **Mathematics for Machine Learning** | Deisenroth, Faisal, Ong | Linear algebra, calculus, probability for ML — specifically designed for ML practitioners | ✅ Free PDF |
| **Introduction to Statistical Learning (ISLR)** | James, Witten, Hastie, Tibshirani | Best approachable stats + ML theory book; R examples but concepts transfer | ✅ Free PDF |
| **The Elements of Statistical Learning (ESL)** | Hastie, Tibshirani, Friedman | The rigorous version of ISLR — graduate level | ✅ Free PDF |
| **Think Stats** | Allen Downey | Statistics through Python programming | ✅ Free online |
| **Think Bayes** | Allen Downey | Bayesian statistics with Python | ✅ Free online |

## Reading Strategy

**Early career (< 2 years ML experience)**:
1. *Hands-On ML* (Géron) — read cover to cover
2. *Natural Language Processing with Transformers* — for NLP/LLM work
3. *Designing ML Systems* (Huyen) — for production awareness
4. *Reinforcement Learning: An Introduction* chapters 1–8 if interested in RL

**Mid career (2–5 years)**:
1. *Designing Data-Intensive Applications* — foundational for system design
2. *Machine Learning Engineering* (Burkov) — practical wisdom
3. Domain-specific deep dives as needed

**Reading tip**: Don't read cover to cover — use these as references. Read the chapters most relevant to your current project. Return and read more deeply when you hit a knowledge gap in practice.
