## Certifications Worth Getting

| Certification | Provider | Cost | Time | Value |
|---|---|---|---|---|
| **TensorFlow Developer** | Google / TensorFlow | ~$100 | 3–6 months prep | Good signal for DL skills |
| **AWS Certified ML Specialty** | AWS | ~$300 | 3–6 months prep | Valuable for cloud ML roles |
| **Professional Data Engineer** | Google Cloud | ~$200 | 3–6 months prep | Cloud data pipeline skills |
| **Deep Learning Specialization** | Coursera | ~$250 | 5 months | Andrew Ng's brand; very recognized |

**Note**: Certifications are a supplement to, not a replacement for, strong project work and GitHub portfolio. Employers care far more about demonstrated skills than certificates.

## Learning Platform Recommendations

| Platform | Best For | Notes |
|---|---|---|
| **Coursera** | Structured specialisations | Free audit for most; good certificates |
| **fast.ai** | Practical deep learning | Top-down approach; excellent community |
| **DeepLearning.AI** | Short, focused AI courses | 50+ free short courses |
| **Hugging Face** | NLP/transformers/audio/RL | Best hands-on ML platform currently |
| **Stanford Online** | Rigorous theory | Video lectures from actual Stanford courses |
| **YouTube (free)** | MIT OCW, 3Blue1Brown, Andrej Karpathy | World-class content for free |
| **Kaggle Learn** | Quick practical intros | Free micro-courses on ML topics |
