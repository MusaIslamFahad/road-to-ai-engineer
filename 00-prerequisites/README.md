# Module 00: Prerequisites

**Phase 0 — Foundation** | Est. time: 2–3 months (full-time) · 4–6 months (part-time)

---

## Learning Objectives

By the end of this module, you will be able to:
- Write clean, idiomatic Python using OOP, functional programming, and advanced features
- Apply linear algebra, statistics, and calculus concepts relevant to machine learning
- Set up a professional Python development environment
- Use Git and GitHub for version control

---

## Prerequisites

- Basic computer literacy — no prior programming or math experience required

---

## Topics Covered

### Python Programming
- Variables, operators, control flow, strings, lists, tuples, sets, dictionaries
- Functions, `*args`/`**kwargs`, closures, higher-order functions
- OOP: classes, inheritance, polymorphism, encapsulation, abstraction, magic methods
- Iterators, generators, decorators, context managers (`with`)
- File I/O: text, binary, JSON, pickle; exception handling
- Time Complexity & Big O notation
- Functional programming: `map`, `filter`, `reduce`, comprehensions

### Essential Mathematics

| Topic | Key Concepts |
|---|---|
| **Linear Algebra** | Vectors, matrices, matrix multiplication, transpose, inverse, eigenvalues, eigenvectors, SVD, tensors |
| **Statistics** | Mean, median, variance, std dev, distributions, hypothesis testing, p-values, Bayes' theorem |
| **Calculus** | Derivatives, partial derivatives, chain rule, gradient descent intuition |
| **Probability** | Sample spaces, conditional probability, independence, expectation |

### Environment Setup
- Python 3.10+ installation (Anaconda recommended)
- Conda and virtual environments
- Jupyter Notebook and JupyterLab
- Git fundamentals: clone, commit, push, pull, branching
- GitHub: repositories, issues, pull requests
- VS Code setup for ML development

## Project: Movie Script Generator

Build a Python CLI app using:
- OOP: `Character`, `Scene`, `Script` classes
- File I/O: load dialogue templates from JSON
- Generators: yield scenes lazily
- Decorators: add logging to methods
